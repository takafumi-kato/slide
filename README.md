# Slide Show

JavaScript を使用して作成した無限ループ対応のスライダーです。

## Features

* 前へ・次へボタンによるスライド切り替え
* 無限ループスライダー
* 3秒ごとの自動再生
* マウスホバー時の自動再生停止
* 現在のスライド番号表示
* CSS Transition を利用したアニメーション

## Technologies Used

* HTML
* CSS
* JavaScript (Vanilla JS)

## Learning Points

このプロジェクトでは以下の内容を学習しました。

* DOM操作
* イベント処理
* setInterval / clearInterval
* CSS Transition
* cloneNode を使用した無限ループスライダーの実装
* 関数分割によるコードの整理
